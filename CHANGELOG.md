# Change log

## 1.0.10

- Added shoe-size, clothing-size, ring-size, paper-size, fuel-volume, and map/drawing-scale converters to Everyday conversions.
- Added coordinate-format, compass-bearing, and rainfall/snow-water converters to Science conversions.
- Added a multi-location time-zone converter using date-specific IANA daylight-saving rules supplied by the browser.
- Created the Time & Dates main category and moved time units, work time, and date difference into it.
- Expanded the searchable toolbox from 92 to 102 tools and from 11 to 12 main categories, with unique icons and badges for every addition.

## 1.0.9

- Added clear subcategories inside every main toolbox category while preserving the four-across main category matrix.
- Added Academics and Culinary as main categories and moved the existing number-skills, probability, cooking-conversion, and recipe-scaling tools into them.
- Renamed Finance & money to Economics without changing its stable category ID or saved-tool compatibility.
- Fixed favorite persistence so deliberately deleted folders and tools—including a completely empty favorites area—stay deleted after reopening the app.

## 1.0.8

- Anchored Rename and Move controls to the top of favorite-folder cards and replaced browser prompts/confirmations with themed create, rename, and delete dialogs.
- Relabeled the main calculator commit key as Save Calculation.
- Repositioned every triangle side and angle value outside the shape and away from its red reference square.
- Rebuilt cylinder and cone front base curves as true half-ellipses matching their top-face perspective in geometry and Element to shape.
- Added apex X and Y coordinates to quadratic equation results.

## 1.0.7

- Added solved side-length values to all three triangle visual labels while retaining the live angle annotations.
- Recalibrated cylinder and cone projections against the adjustable reference cube in geometry and Element to shape, and restored translucent hidden base edges.
- Matched quadratic X/Y units to one common visual scale and applied the same resolution-derived grid and label interval to both axes.
- Raised and separated Rename/Move controls while pinning the red delete button to each folder's upper-right corner.
- Replaced the numbered folder-move prompt with a clickable hierarchical destination tree that excludes invalid descendant targets.

## 1.0.6

- Restored the full-width toolbox panel while keeping the compact orange launcher, and replaced the accordion list with a four-across app-style category matrix and animated category panels.
- Doubled the header logo footprint, repositioned the smaller version badge, and advanced all app/cache version markers.
- Corrected the shared visible-face projection for rectangular prisms and reference cubes, removed duplicate cylinder/cone base outlines, and removed redundant in-canvas reference text.
- Rebuilt triangle annotations as fixed-size, high-contrast labels mapped to the correct three sides and opposing angles.
- Enlarged the quadratic graph, replaced the fixed 0.2 field with a user-entered Graph Resolution, and simplified the grid to thin dark resolution lines plus emphasized zero axes.
- Made tool back navigation return to its launch context: the toolbox matrix, the originating favorite folder, or the calculator.
- Moved folder content below its controls, increased action spacing, and replaced the Delete label with a red close button.

## 1.0.5

- Reduced the toolbox to roughly half its former width while preserving readable, non-overlapping tool names and icons, and compressed the main calculator so the equals key stays in view with the live formula in scientific mode.
- Replaced the flat favorites area with persistent, nameable folders that can be searched, nested, renamed, moved, and deleted; tools can be added within any open folder.
- Rearranged the custom numeric keypad, placing backspace on the top row, subtraction and decimal on the lower rows, and adding a `000` key.
- Replaced reference sliders with compact numeric fields and applied the same editable, common-scale reference system to geometry and element-shape visuals.
- Corrected isometric cube/prism faces, preserved extreme entered proportions, moved reference dimensions below the objects, and labeled triangle legs and angles.
- Rebuilt the quadratic graph as a crisp vector plot with lighter labels, a 0.2-point field, free panning, zooming, a clear curve, and an apex marker.
- Bundled the Science & engineering rocket directly in the interface for reliable GitHub Pages rendering and corrected the rectangle badge to `A=l×w`.
- Added mortgage lifetime loan cost as principal plus lifetime interest plus lifetime PMI.

## 1.0.4

- Enlarged the logo, reduced the version-label emphasis, corrected the divide/backspace positions, and tightened toolbox vertical spacing.
- Narrowed the date-difference inputs and simplified the triangle tool to infer missing sides, angles, and triangle type from the measurements supplied.
- Rebuilt geometry visuals around a common scale so entered proportions and the adjustable red reference square/cube stay accurate without overlap.
- Lightened graph and unit labels, changed quadratic subdivisions to 0.2, added graph zoom, kept the equation on one line, and marked the parabola apex.
- Applied rocket branding to Science & engineering and corrected the rectangle and rectangular-prism equation badges.
- Repaired live currency conversion with direct Frankfurter v2 requests, a direct v1 fallback, and local last-known-rate fallback.

## 1.0.3

- Narrowed the downward-opening toolbox and replaced the environment-dependent wrench with a bundled inline toolbox icon.
- Moved the `v1.0.3` badge beside the logo and prevented double-tap zoom while preserving two-finger pinch zoom.
- Centered the official app title, shortened the introductory copy, removed the redundant expression box, and tightened the calculator display.
- Revised keypad placement, kept the calculator keypad-only on touch devices, and retained the live textbook-style formula and safe partial-expression result.
- Added unlimited favorites, corrected favorite icon alignment, and placed compact Copy and Save controls together.
- Added grouped signed-number input, numeric list keypads, long-unit fitting, calendar and Julian years, km/s and mi/s, and IEC binary results.
- Added the latest-reference-rate currency converter, fixed imperial BMI height, and added an original adult BMI reference chart.
- Added live 2D/3D geometry visuals, a unified triangle solver, polygon names, and a live quadratic graph with whole-number axes and 0.1-unit dots.
- Expanded element-to-shape output and added element density labels plus isometric material visuals for all 118 elements.
- Updated trip fuel, paint coverage, 12-hour work time, exact date difference, mortgage, loan, and other metric/imperial-aware tools.

## 1.0.2

- Added the expanded scientific keypad, blank tool forms, manual history saving, mortgage escrow, unit-system toggles, and the 118-element density calculator.

## 1.0.0
